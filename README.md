# Superstore-Sales-Dashboard

- Project Overview

This project analyzes a 9,994-row retail Superstore dataset using Microsoft Excel. The objective was to clean raw transactional data, perform exploratory analysis, and build an interactive dashboard to uncover key business insights related to sales, profitability, discounts, and regional performance.

- Data Preparation

Corrected data types (dates, currency, percentages)

Validated missing values and inconsistencies

Created derived metrics such as:

Profit Margin

Shipping Duration

Order Year & Month

- Analysis Performed

Using Pivot Tables and aggregations, the analysis focused on:

Overall Sales, Profit, and Order trends

Regional and category-level performance

Discount impact on profitability

Shipping mode comparison

- Dashboard Features

KPI Cards (Total Sales, Profit, Orders, Profit Margin)

Monthly Sales & Profit Trend

Category & Region Performance Analysis

Discount vs Profit Insights

Interactive Slicers for dynamic filtering

- Key Insights

Sales and profit show steady growth with noticeable peaks in later months of the year.

West and East regions contribute higher overall sales compared to Central and South.

Technology generates strong sales and profit compared to other categories.

Higher discount levels are associated with lower profitability.

Standard Class shipping accounts for the majority of sales volume.

- Tools Used

Microsoft Excel (Pivot Tables, Pivot Charts, Slicers, Data Cleaning & Transformation)
